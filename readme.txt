In your own terms, define the following terms:

What is HTTP?

    Hypertext Transfer Protocol is a protocol used for communication between browsers and servers

What is a URL?

    A uniform Resource Locator is an adress that identifies a resource on the internet

What is DNS?

    Domain Name System translates a human readable domain into a server readable ip adress.

What is a query string?

    Part of a URL following a question mark that contains a parameter for request

What are two HTTP verbs and how are they different?

    GET : Used to retrieve data from the server
    POST : Used to send data to the server

What is an HTTP request?

    A request from a browser to server with the HTTP protocol

What is an HTTP response?

A reseponse from server to browser with the HTTP protocol

What is an HTTP header? Give a couple examples of request and response headers you have seen.

Headers give extra info about a request or response



What are the processes that happen when you type “http://somesite.com/some/page.html” into a browser?

    "http://somesite.com/some/page.html" gets turned into a IP adress with DNS
    Browser makes a request to that IP adress
    Server sends a response
    browser builds the page
    browser requests anything else it needs like CSS JS and Images
